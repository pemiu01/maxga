# MaxGa

MaxGa 是由 Flutter 框架制作的漫画在线阅读器，目前 Android 和 IOS 都能正常使用，但是 IOS 暂时并没有任何的上架计划。

目前不提供任何的 **本地漫画阅读** 和 **在线下载功能** , 暂时也没用计划打算做相关功能。
欢迎有打算制作相关功能的开发者发起 PR ，完成部分功能。
